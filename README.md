# Корпоративный веб-портал (ASP.NET Core MVC)

Пример реализации серверной части корпоративного приложения с системой управления контентом и ролевой моделью доступа.

## Технологический стек и архитектура
*   **Backend**: .NET (C#), ASP.NET Core MVC.
*   **Data Access**: Entity Framework Core, SQL Server (MS SQL).
*   **Архитектурные паттерны**: 
    *   **Repository Pattern**: Абстрагирование логики доступа к данным через интерфейсы для повышения тестируемости.
    *   **Dependency Injection**: Использование встроенного контейнера DI для управления временем жизни объектов (DbContext, Repositories).
*   **Security**: Аутентификация и авторизация на основе ролей (Role-based Access Control).
*   **Database**: Использование миграций (Code First) для управления схемой БД.

## Основной функционал
*   **Управление услугами**: Реализован полный цикл CRUD для работы с услугами компании через `IServiceItemsRepository`.
*   **Динамический контент**: Система редактируемых текстовых полей для настройки страниц сайта без изменения кода.
*   **Инфраструктура**: Связь объектов предметной области с базой данных через кастомный `AppDbContext`.

## Инструкция по сборке и запуску
Для развертывания проекта в локальной среде или Docker-контейнере используйте следующие команды:

```bash
# Восстановление зависимостей
dotnet restore

# Сборка проекта
dotnet build

# Запуск приложения с прослушиванием порта 80
dotnet run --urls=http://*:80
